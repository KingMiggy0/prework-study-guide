HTML

  • The head element contains information about the webpage
  • The body Element represents the visible content shown to the user
  • the title elements sets the title to the webpage which appears in the browser's title bar

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • some of the properties and values of CSS are color, font-size, and border

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m: record changes that you have done with a message

JavaScript

  • conditional statements like if, else if, and else are fundamental to programming as they allow you to control the flow of your code based on different conditions.
  • the 'for' loop is essential for working with collections of repeating actions.
